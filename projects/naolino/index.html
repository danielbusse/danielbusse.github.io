NaoLino - Detecting Objects

This project emerged out of the Software Engineering II course of my Bachelor. Here we had to develop an algorithm for a task of choice to implement on the Nao robot. This Project is still (as of 2024) featured on the iCampus website of my university. You can find it here.

My project partner and me chose to double down on the field of image recognition and wanted to implement an algorithm to detect objects. Our Goal was to detect multiple diffent objects with different techniques.

.

The result of the project is mixed. We did get template maching to work easy and relativley reliable. However, we were not be able to train the cascade classifier to detect objects reliably. The results where ambigous and were higly dependent on luck to detect an object correctly. This was due to our lack of experience in the training process. We had not collected enough Data and only used our personal hardware to train the classifer.

We were able to detect objects because we used multiple techniques. We consider this project still a success because we have learned a lot not only about object detection but also on how to approach a unknown and difficult topic.
